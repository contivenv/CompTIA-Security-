# Security+ CertMaster Study (Versão em português) 7.0

As linhas de base de segurança da rede descrevem um conjunto de configurações e controles mínimos de segurança para uma rede. Geralmente, as linhas de base abrangem áreas como configurações de firewall, configurações de segurança de roteadores e switches, configurações de wireless access points e protocolos de segurança para dispositivos de rede. As linhas de base oferecem um ponto de partida para o processo de hardening. Elas garantem que os serviços desnecessários sejam desativados, as senhas padrão sejam alteradas, os protocolos seguros sejam aplicados, e incluem muitas outras configurações. As ferramentas em conformidade com o Security Content Automation Protocol (SCAP) podem automatizar a avaliação das configurações de segurança de acordo com uma linha de base predefinida.

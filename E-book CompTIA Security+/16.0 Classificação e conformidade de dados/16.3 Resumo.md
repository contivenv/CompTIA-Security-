Você deve ser capaz de explicar a importância das políticas e ferramentas de governança de dados para mitigar o risco de violações de dados e de privacidade, bem como implementar soluções de segurança para proteger os dados.

**Diretrizes de privacidade e proteção de dados**

Siga estas diretrizes para criar ou aprimorar políticas e controles de governança de dados:

- Use um modelo de ciclo de vida das informações para classificar e gerenciar os dados confidenciais e pessoais.
- Atribua funções para assegurar o gerenciamento adequado dos dados dentro de seu ciclo de vida.
- Desenvolva classificações para informações confidenciais e pessoais com base em descritores padrão, como público, privado, sensível, confidencial, crítico, proprietário/privativo, PII, informações de saúde, informações financeiras e dados do cliente.
- Use um sistema de gerenciamento de conteúdo que permita a marcação com tags de classificação de arquivos e registros.
- Use produtos de criptografia para garantir a proteção de dados em repouso, em trânsito e em processamento.
- Use programas de capacitação e orientação para ajudar os funcionários a reconhecerem ataques e comportamentos suspeitos.
- Use políticas e procedimentos que impeçam os engenheiros sociais de obterem informações ou acesso não autorizado.
- Use programas de capacitação e orientação para ajudar os funcionários a manterem a segurança e a proteção dos ativos de dados da organização.

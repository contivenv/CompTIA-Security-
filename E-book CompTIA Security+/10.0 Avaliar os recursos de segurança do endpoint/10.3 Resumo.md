

Você deve ser capaz de aplicar políticas e tecnologias de hardening do host e avaliar os riscos de cadeias de suprimentos de terceiros e sistemas integrados/IoT.

**Diretrizes para implementar soluções de segurança do host**

Siga estas diretrizes ao implantar ou reavaliar a segurança e a integração de endpoints com sistemas integrados ou de IoT:

- Estabeleça linhas de base de configuração para cada tipo de host. Garanta que os hosts sejam implantados na linha de base da configuração e configure o monitoramento para garantir a conformidade.
- Configure a criptografia completa do disco.
- Implante uma solução de proteção de endpoint que atenda aos requisitos de segurança para funções como antimalware, firewall, IDS, EDR e DLP.
- Estabeleça procedimentos de gerenciamento de patches para testar atualizações em diferentes grupos de hosts e garanta o gerenciamento do sistema operacional e de softwares de terceiros.
- Crie um plano de gerenciamento para dispositivos de IoT usados no local de trabalho.
- Avalie os requisitos de segurança para sistemas integrados ICS e/ou SCADA:
    - Aquisição de sistemas integrados e IoT seguros
        
    - Uso de controles criptográficos para autenticação, integridade e resiliência
        
    - Controle de acesso e segmentação para redes OT
        
    - Suporte do fornecedor para gerenciamento de patch



Todas as versões de sistemas operacionais para smartphones e tablets, exceto as mais antigas, oferecem criptografia completa do dispositivo. No iOS, existem vários níveis de criptografia.

- Todos os dados do usuário no dispositivo são sempre criptografados, mas a chave é armazenada no dispositivo. Isso é usado principalmente como meio de limpeza do dispositivo. Basta o sistema operacional excluir a chave para tornar os dados inacessíveis; ele não precisa limpar cada local de armazenamento).
- Os dados de email e aplicativos que usam a opção “Proteção de dados” estão sujeitos a uma segunda criptografia usando uma chave derivada e protegida pela credencial do usuário. Isso proporciona segurança dos dados, caso o dispositivo seja roubado. Nem todos os dados do usuário são criptografados usando a opção “Proteção de dados”. Contatos, mensagens SMS e fotos não são, por exemplo.

No iOS, a criptografia da Proteção de dados é ativada automaticamente quando você configura um bloqueio por senha no dispositivo. No Android, há diferenças substanciais nas opções de criptografia entre as versões([source.android.com/security/encryption](https://source.android.com/security/encryption)). A partir do Android 10, não há criptografia de disco completa, pois é considerada prejudicial demais para o desempenho. Os dados do usuário são criptografados por padrão no arquivo.

Um dispositivo móvel contém uma unidade de estado sólido (memória flash) para armazenamento de longo prazo de aplicativos e dados. Alguns aparelhos Android oferecem suporte para armazenamento removível usando mídia externa, como um slot para cartão Micro Secure Digital (SD). Alguns podem ser compatíveis com a conexão de dispositivos de armazenamento baseados em USB. O software de criptografia do sistema operacional móvel também pode permitir a criptografia do armazenamento removível, mas isso nem sempre é o caso. É necessário ter o cuidado de aplicar a criptografia a cartões de armazenamento usando software de terceiros, se necessário, e limitar o armazenamento de dados confidenciais neles.

Um MicroSD HSM é um pequeno módulo de segurança de hardware de fator de forma criado para armazenar chaves criptográficas com segurança. Isso permite que o material criptográfico seja usado com diferentes dispositivos, como um notebook e um smartphone.

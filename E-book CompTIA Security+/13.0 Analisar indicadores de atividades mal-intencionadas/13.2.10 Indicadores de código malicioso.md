Muitos ataques de rede são iniciados por hosts comprometidos que executam vários tipos de código malicioso. Os indicadores de execução de código malicioso são capturados pelo software de proteção de endpoint ou descobertos após o fato nos logs que mostram como o malware interagiu com a rede, o sistema de arquivos e o registro. Para entender como e onde estes indIcadores são gerados, é útil considerar os principais tipos de atividade mal-intencionada:

- **Shellcode**: é um trecho compacto de código usado como payload, projetado para explorar uma vulnerabilidade no sistema operacional ou em um aplicativo legítimo para obter privilégios não autorizados, ou criar um backdoor no host caso seja executado como um trojan. Depois de conquistar uma base de apoio, esse tipo de ataque é seguido por algum tipo de conexão de rede para baixar ferramentas adicionais.
- **Extraçãode credenciais**: o malware pode tentar acessar o arquivo de credenciais (SAM em uma estação de trabalho local do Windows) ou detectar credenciais armazenadas na memória pelo processo do sistema lsass.exe. Além disso, um ataque DCSync tenta enganar um controlador de domínio para replicar sua lista de usuários com suas credenciais com um host não autorizado.
- **Pivoting/movimento lateral/ataque interno**: o procedimento geral é usar a base de apoio para executar um processo remotamente, por meio de uma ferramenta como PsExec ou PowerShell. O invasor pode estar buscando ativos de dados ou pode tentar ampliar o acesso alterando a configuração de segurança do sistema, como abrir uma porta de firewall ou criar uma conta. Se o invasor tiver comprometido uma conta, estes comandos podem se misturar às operações comuns da rede, embora possam ser um comportamento anômalo para essa conta.
- **Persistência**: é um mecanismo que permite que a backdoor do invasor reinicie se o host for reiniciado ou o usuário fizer logout. Os métodos típicos são usar chaves AutoRun no registro, adicionar uma tarefa agendada ou usar assinaturas de eventos do Windows Management Instrumentation (WMI).

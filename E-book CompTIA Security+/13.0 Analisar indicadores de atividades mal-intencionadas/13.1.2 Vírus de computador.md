Um [[vírus]] de computador é um tipo de malware projetado para se replicar e se espalhar de computador para computador, geralmente “infectando” aplicativos executáveis ou código de programas. Existem muitos tipos de vírus, geralmente classificados de acordo com os diferentes tipos de arquivos ou mídias que infectam:

- **Não residente/infectador de arquivos**: o vírus está dentro de um arquivo executável do host e é executado com o processo host. Ele tentará infectar outras imagens de processo no armazenamento persistente e executar outras ações de payload. Em seguida, ele passa o controle de volta para o programa host.
- **Residente na memória**: quando o arquivo host é executado, o vírus cria um processo para si mesmo na memória. O [[processo mal-intencionado]] permanece na memória, mesmo que o processo host seja encerrado.
- **Inicialização**: o código do vírus é gravado no setor de inicialização do disco ou na tabela de partição de um disco fixo ou mídia USB e é executado como um processo residente na memória quando o sistema operacional é iniciado ou a mídia é conectada ao computador.
- **Vírus de script e macro**: o malware utiliza os recursos de programação disponíveis em mecanismos locais de script do sistema operacional e/ou navegador, como PowerShell, Windows Management Instrumentation (WMI), JavaScript, documentos Microsoft Office com código VBA (Visual Basic for Applications) habilitado ou documentos PDF com JavaScript ativado.

Além disso, o termo “multiparte” é usado para designar vírus que usam múltiplos vetores, e “polimórfico” para vírus que conseguem alterar ou ocultar dinamicamente seu código para evitar a detecção.

O que esses tipos de vírus têm em comum é o fato de que eles precisam infectar um arquivo host ou mídia. Um arquivo infectado pode ser distribuído por qualquer meio normal — em um disco, uma rede, como anexo de e-mail ou publicação em rede social ou como um arquivo baixado de um site.

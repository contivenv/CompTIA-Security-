Os primeiros vírus e worms se concentraram no potencial destrutivo da autorreplicação. À medida que os usos lucrativos desse tipo de software se tornaram aparentes, no entanto, eles começaram a ser codificados com payloads projetados para facilitar invasões, fraudes e roubo de dados. Os bloatwares e malwares podem ser usados para diferentes níveis de monitoramento:

- **Cookies de rastreamento**: um cookie é um arquivo de texto simples, não é um malware, mas, se permitido pelas configurações do navegador, os cookies de terceiros podem ser usados para registrar atividades na web, rastrear o endereço IP do usuário e coletar vários outros metadados, como consultas de pesquisa e informações sobre o software e a configuração do navegador. Os cookies de rastreamento são criados por anúncios e widgets de análise incorporados em muitos sites.
- **Supercookies e beacons**: como os softwares dos navegadores dão ao usuário algum controle sobre quais cookies aceitar, as empresas de marketing na web criaram maneiras alternativas e mais difíceis de desativar para implementar o rastreamento. Um supercookie é um meio de armazenar dados de rastreamento de maneira não regular, como salvá-los no cache sem declarar que os dados são um cookie ou codificá-los em solicitações de cabeçalho. Um beacon é uma imagem de pixel único incorporada em um site. Embora invisível para o usuário, o navegador precisa fazer uma solicitação para baixar o pixel e carregar o site, dando ao host do beacon a oportunidade de coletar metadados e impressões digitais do navegador e, potencialmente, executar scripts de rastreamento.
- [[Adwares]]: esta é uma classe de bloatware/PUP (programa potencialmente indesejado) que reconfigura o navegador, por exemplo, permitindo cookies de rastreamento, alterando provedores de pesquisa padrão, abrindo páginas do patrocinador na inicialização, adicionando favoritos e assim por diante. Os adwares podem ser instalados como um programa ou como uma extensão/plug-in do navegador.
- [[Spywares]]: trata-se de um malware que consegue realizar um rastreamento parecido com o dos adwares, mas também monitora a atividade de aplicativos locais, faz capturas de tela e ativa dispositivos de gravação, como microfones ou webcams. Outra técnica de spyware é fazer o redirecionamento de DNS para sites de [[pharming]].
- [[Keyloggers]]: tipo de spyware que tenta ativamente registrar as teclas digitadas para roubar informações confidenciais. O invasor geralmente espera descobrir senhas ou dados de cartão de crédito.

Nota:

Os keyloggers não são implementados apenas como software. Um script mal-intencionado pode transmitir as teclas pressionadas para um site de terceiros. Há também dispositivos de hardware para capturar teclas pressionadas usando um adaptador USB modificado que é inserido entre o teclado e a porta. Esses dispositivos podem armazenar dados localmente ou utilizar conectividade Wi-Fi para enviar dados a um access point secreto. Outros tipos de ataque incluem o uso de detectores wireless para registrar os dados das teclas pressionadas, sobrepor teclados de senha de caixas eletrônicos e assim por diante.

#### Anotações extras

Os vírus e worms tinha um objetivo centrado de autorreplicação e destruição geral do sistema operacional e arquivos. Ao longo dos tempos, os vírus foram se modificando para tornar fraudes, roubos de dados e facilitar invasões usando payloads, que começaram a ser codificado com eles.

Cookies de rastreamento: não é um malware, mas pode ser usado para descobrir localização, IP do computador informações, etc. Tudo isso é disponibilizado no site na primeiro acesso.

Supercookies e beacons: supercookies são formas de um cookie ficar armazenado no cache da memória sem se autodeclarar que é um cookie. Beacon está relacionado a coleta de dados po imagem, pixels que podem conter metadados do usuário.

Adware: abre portas que não estavam lá antes. Redefini configurações padrão de um navegar, coloca páginas de sites na inicialização do seu navegador, adiciona favoritos. Ele pode ser instalado com um software.

Spyware: faz captura de telas, monitora o que está fazendo e existe até um forma de spyware que redireciona o DNS para uma pharming.

Keyloggers: monitora ativamente informações do seu teclado para obter informações sensíveis, como por exemplo, dados de cartão de crédito, documentos sigilosos.

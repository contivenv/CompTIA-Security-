O software do sistema operacional e dos aplicativos pode ser configurado para registrar eventos em log automaticamente. Isso fornece informações valiosas de solução de problemas. Os logs de segurança fornecem uma trilha de auditoria das ações realizadas no sistema, bem como avisos de atividades suspeitas. É importante que a configuração do registro de log e os arquivos sejam tornados invioláveis.
